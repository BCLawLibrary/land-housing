# Publications of the Initiative on Land, Housing & Property Rights

This web application was created for the [Initiative on Land, Housing & Property Rights](https://www.bc.edu/bc-web/schools/law/centers/land-initiative.html#tab-publications) at Boston College Law School. It pulls data from a published [Google Sheet](https://docs.google.com/spreadsheets/d/e/2PACX-1vSvcazCOMPpKONojpgN19KIaqRBEApjYaeyHdB3DKDC9QI8TyWBTMki-vw4fPCHt2HfUQYX-gfj8DUM/pubhtml) using [Papa Parse](https://www.papaparse.com/) and presents them on a webpage using [DataTables](https://datatables.net/) after formatting.
